## 3. Bug runtime THẬT đã phát hiện & fix

- **`GET /api/v1/encounters` → 500** (`System.FormatException: Guid string should only contain hexadecimal characters`). Nguyên nhân: 1 dòng seed có `id` **không phải GUID hex** — `e2eenc01-0000-0000-0000-000000000001` (chữ `n` trong "enc") → EF Core `Guid.TryParse` fail khi materialize → **sập cả danh sách encounters**. Fix: soft-delete dòng seed sai → `/encounters` **200**, Encounter LIST+CREATE **PASS**.
  - *Bài học:* đây là bug mà **đọc code không thấy** — chỉ lộ khi chạy runtime với dữ liệu thật.
