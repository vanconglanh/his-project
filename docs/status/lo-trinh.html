Chuyển đổi schema — Direction B (MySQL)

HOÀN THÀNH ADR 0007 · batch migration 9000–9021

Chuyển nền tảng dữ liệu từ PostgreSQL/RLS sang MySQL 8 + multi-tenant ở application layer để kế thừa schema dump production. EF Core Global Query Filter thay cho Row-Level Security.

  • Drop legacy + dựng lại sec / patient / encounter / labrad / pharmacy / clinic
  • Seed master data + demo, legacy views tương thích
  • Index hiệu năng, migration idempotent (stored proc check information_schema)
  • Multi-tenant enforce ở service + query filter
📌 Nợ kỹ thuật: CHANGELOG/RELEASE-NOTES vẫn ghi PostgreSQL/RLS — cần cập nhật cho khớp source.
